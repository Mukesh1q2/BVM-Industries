# Project Vision

**BVM Engineering Industries Website**

## What This Is
A premium, solutions-oriented corporate website for BVM Engineering Industries, a leading manufacturer of aseptic packaging machinery (FFS, BFS) in India. The site positions BVM as a global authority, shifting from a simple product catalog to a problem-solving partner for pharmaceutical clients.

## Core Value
- **Authority**: Demonstrates 500+ installations and regulatory compliance (FDA/cGMP).
- **Clarity**: Application-led navigation guides users to the right machine for their needs (Injectables, Ophthalmic, etc.).
- **Conversion**: Strategically placed calls-to-action (Brochure Downloads) drive lead generation.
- **Performance**: Optimized for speed (LCP < 2.5s) to serve global audiences effectively.

## Key Features
- **Application-Led Navigation**: Filter machines by use case (Injectables, Respiratory, etc.).
- **Trust Bar**: Visible proof of reliability and scale.
- **Technical Detail**: Comprehensive specs for FFS, BFS, and Cap Sealing machines.
- **Modern UI**: Dark-themed, responsive design with "premium" aesthetics (Sora/Inter fonts, GSAP animations).

## Tech Stack
- **Frontend**: React (Vite), TypeScript, Tailwind CSS
- **Routing**: React Router DOM (HashRouter)
- **Animation**: GSAP, Framer Motion
- **State**: React Context (Theme)
